## Installation

Clone the repository and open the xcode project
Install the iOS app on your iPhone

## Requirements

iOS 10.3
apple developer account for permanent use

## Libraries
- SceneKit
- SpriteKit
- CoreMotion
- UIKit
- AVFoundation
- QuartzCore
